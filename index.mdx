**Motion Icons React** is a powerful library that brings your Lucide React icons to life with smooth, customizable animations. It's designed to be a drop-in replacement for Lucide React icons with zero configuration required.

## Quick Example

Here's how simple it is to add animations to your icons:

```tsx
import { MotionIcon } from '@motion-icons/react';
import '@motion-icons/react/style.css';

function App() {
  return (
    <MotionIcon
      name="Heart"
      animation="heartbeat"
      size={32}
      color="red"
    />
  );
}
```
